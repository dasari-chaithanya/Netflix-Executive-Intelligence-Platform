## Quick Start

```bash
# 1. Clone the repository
git clone https://github.com/yourusername/Netflix-Content-Strategy-Analysis.git
cd Netflix-Content-Strategy-Analysis

# 2. Install dependencies
pip install -r requirements.txt

# 3. Place the dataset
# Download from: https://www.kaggle.com/datasets/shivamb/netflix-shows
# Save to: data/raw/netflix_titles.csv

# 4. Run notebooks in order
jupyter notebook notebooks/

# 5. Launch the dashboard
streamlit run dashboard/streamlit_app.py
```
