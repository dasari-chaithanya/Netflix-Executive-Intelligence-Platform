# Project Pitch & Elevator Pitch

### The 30-Second Elevator Pitch
"I recently built a Netflix Content Strategy Platform. It's an enterprise-grade Streamlit application backed by a custom ETL pipeline and KPI engine. Instead of just showing charts, I engineered an automated storytelling layer that analyzes metrics like 'Catalog Freshness' and generates real-time, plain-English strategic recommendations. It’s fully tested, uses a strict design token system for UI consistency, and is optimized with Parquet for instant load times."

### The 2-Minute Project Pitch (For Career Fairs)
"Hi, I'm [Your Name]. I noticed you're hiring for Data Analysts. A lot of candidates know SQL and Python, but I wanted to demonstrate that I can build production-ready data products. 

For my capstone project, I built a Netflix Content Strategy Platform. I started with a messy dataset of 8,800 titles. First, I built an ETL pipeline to clean and feature-engineer the data, saving it as an optimized Parquet file. 

Then, I built the analytics engine. Rather than just plotting histograms, I defined strict business KPIs—like tracking the 'survival rate' of TV shows or the 'freshness' of the catalog.

Finally, I built a modular Streamlit UI on top. The coolest feature is the Storytelling Engine. When a user filters the dashboard—say, looking only at Indian Comedies—the app doesn't just filter the charts. It recalculates the KPIs, checks them against predefined thresholds, and prints out a strategic recommendation for the executive, telling them whether to increase or decrease investment in that segment. 

It taught me a lot about data contracts, software architecture, and bridging the gap between raw data and business value."
